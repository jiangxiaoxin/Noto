Editor options

Noto allows you to pick any installed font to use as the editor font. If you are handling source code and/or HTML files, it is recommended to use a mono-spaced font. However you can choose a proportional font if you wish.

  1. In Noto, choose Noto > Preferences….

  2. On the Preferences panel, select the "Editor" tab.

The following controls are visible in this panel:

Font

The font used in the editor.

To choose a font family and size:

  1. Click “Choose Font” to open the Font Panel.
  2. Navigate to the desired font and choose a point size.
  3. Close the Font Panel.

Options

Disable smart substitutions by default.

Enabling this checkbox will stop the automatic substitution of quotes, dahses, and other characters into prettier “typist” characters.

These characters are very often used in source code files, and the substitutions might brake the source code.

Disable automatic spelling checker by default.

Enabling this checkbox will stop the highlighting and automatic substitution of misspelled words.

Insert spaces instead of tabs.

Enabling this checkbox will cause the editor to insert space characters instead of tab characters when the tab key is pressed.

Counts white-spaces in total characters.

Enabling this checkbox will cause white-spaces (which include spaces, tabs, and new line characters) to be counted in the total number of characters displayed in the Info Bar. Read more about the Info Bar.

Keep indentation on new lines.

Enabling this checkbox will cause the editor to automatically insert indentation matching the previous line on the new line created after the return key is pressed.

Tab Size

The number of characters used as the width of a tab character, and thus also the width used for a single indentation.

Info Bar

Switches between the different available modes of the Info Bar:

  • Disabled:
    Hides the Info Bar.

  • HUD:
    Shows the Info Bar as a floating semi-transparent area over the editor text.

  • Status Bar:
    Shows the Info Bar as an opaque area on the bottom of the editor.

Read more about the Info Bar.
