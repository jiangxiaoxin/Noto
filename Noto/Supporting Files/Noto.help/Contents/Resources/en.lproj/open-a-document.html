Open a document

Noto can open any plain text documents, such as a simple TXT file, CSV, markdown, HTML, CSS, and other source code files such as Swift, C, or Java.

Open a document

  • In Noto, choose File > Open, select the document, then click Open.

    If your document is stored in iCloud Drive, you can select Noto in the iCloud section of the Finder sidebar, then double-click your document. For more information about working with files in iCloud, see Store documents with iCloud Drive.

Reopen the document specifying a different encoding

Sometimes Noto can fail to recognize the correct encoding for a document. This usually happens when the document has several ASCII-equivalent characters and very few characters from other encodings, such as it can’t be sure which encoding is the correct one. If that is the case, follow the instructions below to reopen the document with a different encoding.

  1. Choose File > Reopen with Encoding….

  2. Choose the desired encoding to use while reopening the file, then choose OK.

The document contents will be reloaded from the file using the specified encoding. If that is not possible, Noto will present the encoding picker dialog once again. If you can’t seem to be able to correctly reopen a file with the correct encoding, that file might be corrupted.
