Change a document’s encoding

You can save a document using a different encoding than the one used for reading it.

Open Noto for me

  • Note: If the document you opened doesn’t look right, try reopening the document with a different encoding.

    For more information about opening/reopening documents, see Open a document.

Saving a document with a different encoding can be useful if you wish to open that file with another device or software that’s not compatible with certain encodings (for example an older device or software).

For general usage on macOS, UTF-8 is commonly recommended as the best encoding to use for all languages and scripts, and is the default encoding used by Noto.

  1. In Noto, choose File > Save with Encoding…

  2. Choose the desired encoding to use while saving the file next, then choose OK.

  3. Name the document, add one or more tags, and then select a location.

    To save your document in the Noto folder in iCloud Drive, click the Where pop-up menu, then choose “Noto — iCloud.” If you choose iCloud Drive in the pop-up menu, your document is saved in iCloud Drive, but not in the Noto folder. If you don’t see the pop-up menu, click Noto in the iCloud section of the Finder sidebar.

  4. Click Save.
