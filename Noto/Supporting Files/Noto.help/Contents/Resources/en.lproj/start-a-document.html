Start a new document

Noto is focused on the creation and editoring of plain text documents. Plain text documents don’t have formatting options, such as fonts and colors. Instead, the display and appearance of the documents is provided by the application, and is applied to all open documents. This provides for a lightweight text file and wide compability with other text editoring applications in all platforms.

Open Noto for me

  1. Open Noto to start a new document.

    If the document browser dialog appears, you have iCloud Drive turned on. If you want your new document to be saved in the Noto folder in iCloud, select Noto in the iCloud section of the Finder sidebar, then click New Document. For more information about working with files in iCloud, see Store documents with iCloud Drive.

    If Noto is already open, choose File > New (from the File menu at the top of your screen) to start a new document.

  2. When the document opens, begin typing.

  3. Your document saves automatically as you work. You can also do any of the following:

    • Print a document: Choose File > Print.

    • Save the document with a spacific encoding: Choose File > Save with Encoding… Read more.
